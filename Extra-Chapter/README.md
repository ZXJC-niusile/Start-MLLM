# Extra-Chapter

`Extra-Chapter` 用来放置正文之外、但和多模态工程落地强相关的扩展专题。

这一部分的定位不是重复主章节，而是补主线之外最常见、最容易踩坑的几个方向。当前已包含：

- [Extra01 OCR 与文档理解专题](./Extra01-OCR与文档理解专题.md)
- [Extra02 多图输入与比较专题](./Extra02-多图输入与比较专题.md)
- [Extra03 长图处理与切块策略专题](./Extra03-长图处理与切块策略专题.md)

建议阅读顺序：

1. 如果你做截图、票据、合同、表格相关应用，先看 `Extra01`。
2. 如果你要做商品对比、前后变化分析、质检复核，先看 `Extra02`。
3. 如果你的输入经常是长截图、长网页、长文档，再看 `Extra03`。

与正文穿插时，可优先对照：**理论篇** [第二章](../docs/chapter2/第二章%20视觉编码器与跨模态对齐.md)（分辨率与 token）、[第四章](../docs/chapter4/第四章%20数据、训练与微调.md)（OCR 类样本）；**实战篇** [第五章](../docs/chapter5/第五章%20评测体系与工程选型.md)（按场景建集）、[第六章](../docs/chapter6/第六章%20推理部署与%20Serving.md)（预处理与长图策略）。全书分册与依赖表见 [前言](../docs/前言.md)。
