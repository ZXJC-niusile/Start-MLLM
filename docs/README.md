# Start-MLLM

欢迎来到 `Start-MLLM`。

这是一份面向中文学习者的多模态大模型教程，默认采用“先搭框架，再做实践”的学习顺序。你可以把它理解为一条从 `LLM 入门视角` 走向 `MLLM 工程视角` 的路线图。

从这一版开始，章节内容尽量统一成四段：

1. 主线概念
2. 工程要点
3. 实战衔接
4. 章末练习

## 这份教程解决什么问题

很多人学多模态时会卡在三个位置：

1. 论文名词很多，但不知道每个模块在系统里扮演什么角色。
2. 知道模型能“看图说话”，但不知道数据、训练、部署分别怎么落地。
3. 会调用 API，但不会把多模态能力接入自己的应用或 Agent。

本项目就是围绕这三个问题展开。

建议阅读方式：

- 先读主线，不要一开始深陷细节。
- 每学完一章，立即做“章末练习”中的 1 到 2 个任务。
- 第四章、第五章、第七章、第八章、第九章都配了更偏实战的脚本或脚手架，适合作为你自己项目的起点。
